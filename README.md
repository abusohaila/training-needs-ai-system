# نظام الذكاء الاصطناعي لتحليل الاحتياجات التدريبية

نسخة ويب مبسطة ضمن مشروع «نحو تعلم ذكي ومستدام».

## التشغيل محليًا للاختبار
```bash
pip install -r requirements.txt
streamlit run app.py
```

## النشر على Streamlit Community Cloud
1. ارفع ملفات المشروع إلى مستودع GitHub.
2. افتح Streamlit Community Cloud.
3. اختر المستودع وملف `app.py`.
4. انشر التطبيق.

المستخدم النهائي يحتاج فقط إلى رابط التطبيق ومتصفح إنترنت.
